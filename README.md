## üìÇ Project Overview

This repository contains two single-page web applications, which are a Biodata form and A calculator.

### 1. Matrimonial Biodata Form (`biodata.html`)
A responsive biodata submission form.
* **Key Features:**
    * **Validation:** Utilizes HTML5 native attributes (`required`, `pattern`, `min`, `max`) for client-side validation without relying on heavy scripts.
    * **Styling:** Custom CSS for a professional, centered layout using Flexbox.
    * **Interactivity:** JavaScript event listener to handle form submission and display a browser alert upon success.
    * **Input Types:** Demonstrates usage of `text`, `number`, `radio`, `select`, `tel`, and `textarea`.

### 2. Standard Calculator (`calculator.html`)
A functional calculator with a dark-mode UI.
* **Key Features:**
    * **Layout:** Uses `display: grid` to create a responsive button layout.
    * **Logic:** Implements JavaScript `eval()` function for arithmetic operations and error handling (try-catch block).
    * **Design:** Modern dark aesthetic with distinct colors for operators and actions.
